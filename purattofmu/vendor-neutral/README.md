# ベンダー中立型統合レイヤー

## 概要

翻訳ラボは、特定のTMSやMTエンジンを推奨しません。顧客にとって最適なツールの組み合わせを、自由に選択・変更できる基盤を提供します。

---

## 統合アーキテクチャ

```
翻訳ラボ プラットフォーム（中立レイヤー）
    ↓ API統合
├─ TMS（Phrase、Smartling、Memsource等）
├─ MT（DeepL、Google、Claude、GPT等）
├─ 用語管理（TermBase、SDL等）
└─ 品質チェック（QA tools各種）
```

---

## 主な機能

- **主要TMSとのAPI接続テンプレート** — 主要ツールとの統合を素早く実現
- **MTエンジン切り替え機構** — 用途に応じてMTエンジンを柔軟に変更
- **用語管理DB統合** — 既存の用語資産をそのまま活用
- **品質メトリクス収集基盤** — 各ツールの品質データを一元管理

---

## 差別化要素

ベンダーに依存せず、エンタープライズが「最適なツールの組み合わせ」を自由に選択・変更できる基盤を提供します。翻訳ラボ自身がベンダーロックインを生まないこと——これが信頼の基本です。
