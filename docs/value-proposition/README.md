# LTPとしてのポジショニング

## 翻訳ラボとは何か

```
翻訳ラボ ≠ 翻訳サービスプロバイダー
翻訳ラボ = AI翻訳ワークフロー設計プラットフォーム
```

翻訳ラボは、\*\*Language Technology Platform（LTP）\*\*です。

従来の言語サービスプロバイダー（LSP）が「翻訳成果物の提供」に焦点を当てるのに対し、翻訳ラボは**ワークフロー設計基盤そのもの**を提供します。

***

## 提供価値の核心

SlatorConをはじめとする業界イベントでエンタープライズ担当者が語ることは、一貫しています。

> **「AIツールが欲しい」のではない。**\
> &#xNAN;**「AIツールをどう組み合わせ、どう統治するかの設計図が欲しい」**

この設計図を、可視化し、構造化し、カスタマイズ可能なテンプレートとして提供する。それが翻訳ラボのコア・バリューです。

***

## 6つの提供価値

1. **ワークフロー可視化** — 全体を一目で把握できるようにする
2. **AI統合ポイント最適化** — どこにAIを入れるかを設計する
3. **品質管理プロセス構造化** — 担当者依存から仕組み依存へ
4. **ワークフロー設計方法論の提供** — 設計できる組織をつくる
5. **ベンダー中立性** — ツール選択の自由を守る
6. **ナレッジ蓄積基盤** — 組織の学習を資産に変える

***

## 3つの約束

```
翻訳ラボは、翻訳を売らない。
ワークフロー設計能力を売る。

翻訳ラボは、ツールを売らない。
ツールを組み合わせる方法論を売る。

翻訳ラボは、答えを売らない。
答えを見つける枠組みを売る。
```
