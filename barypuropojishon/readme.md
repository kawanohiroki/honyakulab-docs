# このドキュメントについて

> **翻訳ラボは、翻訳を売らない。ワークフロー設計能力を売る。**

***

## このドキュメントについて

このドキュメントは、**翻訳ラボ**が Language Technology Platform（LTP）として提供する価値と、その技術的・方法論的基盤を記述したものです。

AI翻訳時代において、エンタープライズが直面する課題は「翻訳品質」ではなく、「**AI翻訳ツールをどう組み合わせ、どう統治するかの設計図**」の不在です。

翻訳ラボは、この設計図を——可視化し、構造化し、カスタマイズ可能なテンプレートとして——提供します。

***

## 翻訳ラボとは

翻訳ラボは、従来の言語サービスプロバイダー（LSP）でも、翻訳管理システム（TMS）ベンダーでもありません。

```
翻訳ラボ ≠ 翻訳サービスプロバイダー
翻訳ラボ = AI翻訳ワークフロー設計プラットフォーム
```

立教大学・山田優教授との産学連携のもと、30年以上の翻訳業界知見と最新のAI技術実装能力を組み合わせた、業界唯一のポジションから価値を提供します。
